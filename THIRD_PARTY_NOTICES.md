# Third-party notices

FlagCraft is an independent fan-made project and is not affiliated with Mojang or Microsoft.

## Golden Apple image

The interface uses the Golden Apple image supplied by the project owner from Minecraft Wiki. A local pixel-art Golden Apple fallback is included so the layout remains usable if the external image cannot load.

Source: https://minecraft.wiki/w/Golden_Apple

## Diamond image

The main-menu Achievements card uses the Diamond image supplied by the project owner from Minecraft Wiki. A local original pixel-style diamond fallback is included for offline/error handling.

Source: https://minecraft.wiki/w/Diamond

## Flags

Country flag images are requested at runtime from FlagCDN. If an image cannot load, FlagCraft displays an emoji fallback so gameplay can continue.

## Sound effects

The simplified sound set contains only two optional runtime effects from Myinstants:

- Interface click
- Level-up

Correct-answer, wrong-answer, combo/totem, hurt, and advancement/challenge-complete effects are not used. If a remote clip is not ready, FlagCraft immediately plays a small Web Audio fallback instead. Sounds can be disabled in Settings.

Source pages:
- https://www.myinstants.com/en/instant/minecraft-click-70367/
- https://www.myinstants.com/en/instant/minecraft-level-up-sound-83487/
